# 🔬 Simulation and Modeling Lab – BSc.CSIT Fifth Semester  

This repository contains implementations of **continuous and discrete system simulations**, **random number generation techniques**, and **statistical testing methods**.  
All programs are implemented in **Python** for clarity, modularity, and ease of testing.  

## 📁 Lab Tasks  

### System Simulations  
1. Continuous System Simulation – Chemical Reactions  
2. Discrete System Simulation (Monte Carlo Method: Estimating π)  
3. Single Queue Server System Simulation (M/M/1 Model)  

### Random Number Generation  
4. Discrete Random Variate Generation  
5. Linear Congruential Method  
6. Multiplicative Congruential Method  
7. Mid-Square Method  

### Markov Chains  
8. Weather Prediction Simulation  

### Statistical Testing  
9. Kolmogorov-Smirnov (KS) Test for Uniformity  
10. Chi-Square Test for Uniformity
