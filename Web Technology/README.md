# Web Technologies Lab Solutions (BSc CSIT 5th Semester)

This repository contains solutions for the Web Technologies Lab
questions.

## Questions Covered

1.  Meta tags for XYZ Travel Agency.
2.  About Us webpage.
3.  Signup form with required fields.
4.  External CSS integration.
5.  JavaScript form validation.
6.  XML file for students.
7.  XPath example.
8.  Login authentication using PHP and MySQL.
9.  Remember Me feature.
10. Store signup data using PHP and MySQL.
11. Retrieve and sort records by age.
12. Edit and Delete records with JavaScript confirmation.

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript
-   PHP
-   MySQL
